Bug fixes galore!

- Packet size crashes and errors should now be fixed! You should no longer need other mods to fix any of these issues also!
- Items purchased from the admin shop that should not be stacked are no longer stacked
- Fixed crash caused by threshold switch from Create
- Various formatting and rendering fixes
{/* truncate */}
Improvements:

- Manifests now give you your items back so they can be used in survival mode if desired. This can be toggled in the config
- No longer display OP messages when creating manifests if in survival mode
- [/adminshop command](/docs/1.20.1/admin-tutorial/server-config/admin-shop-command) can now be enabled for all players via the [config file](/docs/1.20.1/category/server-config)
- Formatting and rendering improvements for display texts and GUI elements
