# Exporter

The exporter allows you to **automatically** sell items using **energy.**

<img
	src={block}
	width="300"
	style={{
		'border-radius': '10px',
	}}
/>

## Get started

The exporter requires a manifest to function and without it, it is **useless.**
Once you have acquired a **suitable manifest,** simply put it in the **manifest slot** and the exporter will start to sell **any items listed on the manifest.**
If the manifest is **golden** the exporter well be able to sell **any sellable** item.

### Power

You can **power** the exporter using **FE** (Forge Energy), you can use **any mod** to get the energy to the exporter.
The exporter can be powered from **any side.**
